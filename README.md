Egg Catching Game is a simple 2D game developed using Python and Pygame.
In this game, an egg falls from the top of the screen and the player controls a basket to catch it.
Each time the player catches the egg, the score increases.
If the egg falls down without being caught, it resets and falls again.

This project helps beginners understand basic game development concepts like animation, keyboard controls, collision detection, and scoring system.
